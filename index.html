Projects

This project was focused on using cluster analysis with a much broader application than the filtering of empirical covariance matrices from the estimated correlation coefficients. That is, using clustering over an empirical variance-covariance matrix for diversification, i.e., to quantify the risk associated with the portfolio. And to be able to do that, we need to obtain distances between the financial time series. So it all boils down to find a suitable distance measure based on (copula, margins) representation between financial time series which when used in clustering procedures, can encode all the available information present in the cross-dependent random variables.

Tools/techniques: Python, Scikit-learn, Numpy, Pandas, Matplotlib, Statistical Analysis, Cluster Analysis etc.

Generated an image mosaic/ panorama from images. Given a set of image correspondences, the projection transforms were calculated and the final mosaic image was generated.

Adjacent pictures that had atleast 40% overlap were ordered from left to right. Generated SIFT features among images, then established correspondence between features. Then applied RANSAC (Random Sampling and Consensus) to get rid of outliers. Then generated initial estimate of Homography using inlier correspondence points. Then Refine the Homography estimate using Levenberg- Marquardt optimization. Repeated the above steps for each of the adjacent image pairs. After obtaining Homographies for each of the picture pairs, get the homographies with respect to the central image. Then projected all images (using inverse warping) on to a blank canvas.

Tools/techniques: SIFT, RANSAC , Inverse Warping,Python.
Link: https://github.com/Nisha925/Image-Mosaicing
